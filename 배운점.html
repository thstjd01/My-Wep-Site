맺는말

<배운점>

워드로 이미지 삽입, 표 만들기, 글자 크기와 디자인 등을 고려하여 사용해 보면서 여러가지 기능을 배울 수 있는 기회가 되었고,
웹 페이지 내용을 미리 워드로 내용을 정리하면서 나만의 웹 페이지를 상상함으로써 웹 프로그래밍을 열심히 배우겠다는 자극을 받았습니다.

웹 페이지의 디자인을 위해 css3에 있는 여러가지 색을 찾아보고 페이지에 넣어보면서 다양한 색을 경험해 볼 수 있었고,
더 많은 기능들을 배워서 웹 페이지를 기술적으로나 미적으로 한층 더 업그레이드 하고 싶다는 생각이 들었습니다.

Javascript에서 c언어와 비슷한 코드, 객체, HTML DOM, document, 이벤트, 캔버스 그래픽 등을 배웠습니다.
HTML이나 css3 보다 Javascript가 더 어려웠지만 강의를 듣고 연습문제와 나만의 웹페이지 만들기를 통해 Javascript와 친숙해 질 수 있었습니다.
이번 웹 개발 입문 강의를 통해 웹 프로그래밍에 친숙해 졌고 재미를 느껴, 수업이 종강해도 따로 웹 프로그래밍을 더 깊게 배워보고 싶다고 생각했습니다.
